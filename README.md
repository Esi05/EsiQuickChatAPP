# ChatApp - PROG5121 Programming 1A

## Description
ChatApp is a console-based Java messaging application developed 
for the PROG5121 Programming 1A module. The application allows 
users to register, log in, and send messages to other users. 
Messages are validated, assigned a unique ID and hash, and 
stored in a JSON file for record keeping.

## CI Status
![Java CI with Maven](https://github.com/YourUsername/ChatApp/actions/workflows/testJava.yml/badge.svg)

## Features
- User registration with username and password validation
- Secure login system with error handling
- Send up to a user-defined number of messages
- Message ID and hash generation for each message
- Options to send, store, disregard, or return messages
- JSON file storage for all sent messages
- Automated JUnit testing with GitHub Actions CI pipeline
